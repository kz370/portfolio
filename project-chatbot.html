AI Chatbot SaaS: Built From Scratch.

A complete FilamentPHP admin dashboard for managing AI chatbot deployments, user quotas, and a custom WordPress plugin for easy embedding.

End-to-End Development

I developed this entire platform from start to finish. The core is a FilamentPHP/Laravel admin dashboard that allows admins to manage chatbot users, track usage quotas, and configure AI settings.

Key Deliverables:

  • FilamentPHP admin panel for user and subscription management.
  • Quota tracking system with usage limits and billing hooks.
  • Custom WordPress Plugin allowing clients to embed the chatbot widget on their sites with a simple shortcode.
