The Client

Canada in Egypt.

CIC is the first provider of Canadian education in Egypt through its partnership with Cape Breton University. Its first-ever campus was established in 2004 in New Cairo with the aim of providing a world-class education to its students.

The Challenge

Fragmented Data, Operational Chaos.

Prior to this system, the university operated on disjointed islands of data: WordPress for the front, legacy SQL for grades, and external providers for payments.

My Role: I designed and built the clean, high-performance Laravel API that acts as the backbone for the Flutter mobile app. This entails managing secure authentication, real-time grade syncing, and payment gateway integration.

Core Features Implemented

  • Unified API Gateway

    A solid Laravel REST API serving both the Flutter Mobile App and Vue.js Web Dashboards, ensuring data consistency.

  • Logistical Intelligence

    Automated shipping workflows. Students request documents → System calculates fees → API dispatches to Courier.

  • Secure Payments

    Deep integration with Klickit for tuition processing, handling webhooks and instant transaction verification.

}
