Package 01

Krayin Multi-Tenancy

A production-ready package that brings Database-per-Tenant architecture to Krayin CRM with isolated databases per organization.

Current Scope: automated tenant database creation, landlord/tenant connection handling, tenant-specific migration flows, and subdomain-based tenant access.

// Install + tenant migrations composer require kz370/krayin-multi-tenancy php artisan vendor:publish --tag=krayin-multi-tenancy-config php artisan vendor:publish --tag=krayin-multi-tenancy-migrations // Landlord setup php artisan migrate php artisan krayin-crm:install // Keep tenant schemas updated php artisan tenants:migrate
Package 02

Laravel JWT Auth

A full-featured Laravel package for JWT authentication using access + refresh tokens and secure token lifecycle controls.

Includes refresh-token hashing, rotation, replay detection, multi-guard support, and per-device session management for modern API backends.

View on GitHub
// AuthController.php public function login(Request $request) { $credentials = $request->only('email', 'password'); $tokens = JwtAuth::attempt($credentials, 'MacBook Pro'); if (!$tokens) { return response()->json(['error' => 'Unauthorized'], 401); } return response()->json($tokens); }
